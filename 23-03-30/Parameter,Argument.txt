Q: Parameter와 Argument의 차이에 대해 설명해주세요.
A: Parameter와 Argument는 프로그래밍에서 함수나 메서드와 관련된 용어로, 종종 혼동되기도 합니다. 이 두 용어의 차이점은 다음과 같습니다.

Parameter (매개변수):
Parameter는 함수나 메서드의 선언부에서 사용되는 변수입니다. 함수나 메서드가 호출될 때 값을 전달받기 위해 사용되며, 해당 함수나 메서드 내부에서만 접근 가능합니다. 함수 선언 시, 괄호 안에 정의된 변수들이 매개변수입니다. 예를 들어, 아래의 함수에서 a와 b는 매개변수입니다.
java
Copy code
public int add(int a, int b) {
    return a + b;
}
Argument (인수):
Argument는 함수나 메서드를 호출할 때 전달되는 실제 값입니다. 이 값들은 함수나 메서드의 매개변수에 할당되어 사용됩니다. 예를 들어, 아래의 코드에서 3과 5는 인수입니다.
java
Copy code
int result = add(3, 5);
요약하면, Parameter는 함수나 메서드의 선언부에서 사용되는 변수로, 값을 전달받기 위해 사용됩니다. 반면, Argument는 함수나 메서드를 호출할 때 전달되는 실제 값으로, 해당 함수나 메서드의 매개변수에 할당됩니다.
